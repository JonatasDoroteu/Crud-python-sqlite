# 🐍 CRUD Python + SQLite

Projeto simples de CRUD (Create, Read, Update, Delete) desenvolvido em Python utilizando SQLite como banco de dados.

---

## 📌 Objetivo

Praticar conceitos de programação com Python, incluindo:
- Operações de CRUD
- Uso de banco de dados SQLite
- Organização de código em arquivos separados
- Lógica de programação

---

## ⚙️ Tecnologias utilizadas

- Python 3
- SQLite3
